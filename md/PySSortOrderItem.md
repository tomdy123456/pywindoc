# PySSortOrderItem


## PySSortOrderItem Object

An item in a SortOrderSet\.

#### Parameters

  - propTag : int

    A property tag\.

  - order : int

    The order in which the data is to be sorted\. Possible values are: 

mapi\.TABLE\_SORT\_ASCEND, mapi\.TABLE\_SORT\_COMBINE and mapi\.TABLE\_SORT\_DESCEND
